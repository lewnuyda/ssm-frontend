## ⚙️ Installation & Setup (React App)

### **1. Clone the repository**

```bash
git clone https://github.com/lewnuyda/ssm-frontend.git
cd frontend
```

### **2. Install dependencies**

```bash
npm install
# or
yarn install
```

### **3. Run the development server**

```bash
npm run dev
# or
yarn dev
```
